# 林间阅读 · 设计规范 v1.0

> 对齐 Apple Human Interface Guidelines 的精神（清晰 Clarity / 遵从 Deference / 纵深 Depth），
> 结合阅读类产品的排印传统。本文件是唯一视觉事实来源；代码中的落地位置以「📍」标注。
> 说明：v1.0 之前的界面为功能优先的迭代产物，无成文规范；本文件自 v0.1.7 起生效并逐步收敛存量页面。

---

## 1. 视觉主题与氛围

**关键词：林间、纸感、安静。**

阅读应用的界面要「让位于内容」（HIG: Deference）——色彩克制、装饰极少，
让正文与书籍封面成为画面中唯一的主角。氛围基调是自然材质的：暖白纸、森林绿、原木棕，
拒绝高饱和撞色与深色大色块。任何新增视觉元素先问一句：它是在帮用户读书，还是在抢戏？

## 2. 颜色体系

### 2.1 品牌色

| 名称 | 色值 | 用途 |
|---|---|---|
| 松绿 Primary | `#2E6B4F` | 主操作、选中态、✦ 锚点、进度 📍`main.dart seed` |
| 暖白纸 Surface | `#F7F6F2` | 全局浅色背景 📍`scaffoldBackgroundColor` |

派生色阶由 Material 3 `ColorScheme.fromSeed` 从松绿生成（保证对比度与暗色模式一致性），
不手工发明中间色。

### 2.2 阅读纸张（内容区专用）

阅读区背景独立于全局主题，八种纸张成套「背景+文字」配色 📍`reader_papers.dart`：
纯白 / 暖白 / 羊皮纸 `#F5ECD9` / 豆沙绿 `#E3EDD9` / 青灰 `#E8EDF0` / 牛皮纸 `#E9DAC0` /
夜间黑 `#17191C` / 夜蓝 `#16202B`。暗色纸张联动全局暗色主题。文字色不允许纯黑（用 `#1F2328` 级墨色）。

### 2.3 功能色

封面色板七套自然色系（松绿/枯木棕/山雾蓝/陶土/暮紫/湖青/果酱红）📍`shelf_screen.dart _coverPalettes`，
按书名散列分配，同一本书永远同色。高亮三色（黄/绿/蓝，40% 透明度）📍`highlightColors`。
错误/警示使用 scheme 派生色，不自定义红色。

## 3. 字体规范

| 场景 | 字体 | 规格 |
|---|---|---|
| 界面（Apple 平台） | 系统 San Francisco 📍`fontFamily: .AppleSystemUIFont` | 跟随系统动态字重 |
| 界面（Android） | 系统默认（Roboto/MiSans） | — |
| 正文阅读 | 衬线：Songti SC → STSong → Noto Serif SC 📍`serifFallback` | 默认 18pt / 行高 2.0 / 字距 +0.2 |
| 书名（封面/大标题） | 同衬线栈，w700 | 书架大标题 28pt；封面 15pt |

层级（对齐 HIG Typography）：大标题 28 / 页标题 24 / 区块标题 17 semibold /
正文 15 / 辅助 13 / 说明 12 / 角标 10。**任何界面文字不小于 10pt。**

**行长规则（阅读学核心约束）**：正文栏最大宽度 720px（约 35–40 汉字/行）📍`reader_screen.dart ConstrainedBox`，
设置/表单类页面最大 560px，书架内容区最大 1120px。

## 4. 间距系统

**4pt 网格**：所有间距取 4 的倍数，常用档位 4 / 8 / 12 / 16 / 24 / 32。

| 场景 | 值 |
|---|---|
| 页面边距（桌面） | 32 |
| 卡片/面板内边距 | 16–18 |
| 区块间距 | 24–28 |
| 行内元素间距 | 8–12 |
| 书墙卡片间距 | 横 22 / 纵 26 |
| 正文段落间距 | 14 |

## 5. 组件规范

- **圆角**：小件（输入框/按钮）8；卡片/封面 12；胶囊（操作条/搜索框）20–24；对话框 28（M3 默认）。
- **分割线**：0.5pt 发丝线 📍`dividerTheme`，禁用 1pt 以上粗线。
- **阴影**：仅两档——封面 `blur 10 / y4 / 品牌色 35%`；浮层 `elevation 6`。平面元素零阴影。
- **按钮**：主操作 FilledButton（松绿底白字）；次操作 OutlinedButton；行内 TextButton。一屏至多一个主按钮。
- **输入框**：OutlineInputBorder，聚焦色为松绿；紧凑场景（头部搜索）用无边框填充式 + 圆角 20。
- **书籍卡片**：固定宽 168、比例 0.62；渐变封面 + 左侧书脊线（白 28%）+ 左下作者/格式角标；进度条 3pt 置于卡下。
- **窗口（macOS）**：默认 1180×800，最小 880×620，居中 📍`window_manager`。

## 6. 动效规范（L2）

| 场景 | 动效 | 时长/曲线 |
|---|---|---|
| 页面切换 | 共享轴水平推入 📍`SharedAxisPageTransitionsBuilder` | 300ms 默认 |
| 解释侧栏 | 右侧滑入 + 淡入 | 260ms easeOutCubic |
| 划选操作条 | 底部滑入 + 淡入 | 220ms easeOutCubic |
| ✦ 锚点入场 | 弹性缩放 0.4→1 | 420ms elasticOut |
| 卡片悬停 | 缩放 1.045 | 160ms easeOut |
| 步骤指示器 | 宽度渐变 | 240ms |

原则：动效只出现在「状态发生变化」的时刻，时长 150–450ms 区间，全局曲线统一用 easeOutCubic 系，
不做循环动画、不做纯装饰动画（HIG: 动效服务于理解）。

## 7. 语言与文案

界面动词化短语（「导入书籍」而非「书籍导入功能」）；错误信息必须包含「发生了什么 + 用户能做什么」；
不使用感叹号堆砌；技术词汇出现时附一句人话解释。

## 8. v1.1 增补：排版与规范骨架（借鉴 Linear/Material/Ant 系设计系统，保留松绿品牌）

对照业界规范做的一轮「骨架借鉴」——只借结构性规则，不改品牌色与阅读纸色体系。

**字重三档（中文 UI 实践）**：全局仅用 400 / 500 / 600。中文加粗位是 500（苹方无 Bold，
Ant Design 中文同此实践）；600 仅用于英文/数字加粗与大标题；禁 700+（中文过重，苹方也不提供）。
正文内粗体（EPUB `<b>/<strong>`）渲染为 600。

**正文富文本档**（阅读器）：标题分级 h1/h2/h3 = 正文字号 +8/+5/+3、行高 1.4、字重 600；
引用块 = 左侧 3px 竖线 + 前景色 78% 透明度；插图居中、圆角 8、限高 320。
分页测量与渲染共用同一套样式常量（`paginator._styleFor`），改字号必须两处同步。

**对比度**：正文/标题走主题前景色（AA 以上）；次级文字统一 `onSurfaceVariant`，
禁止直接用低对比灰做业务文本；纯装饰字符除外。

**层级用描边不用漂浮**：卡片默认 border 划界；阴影只留封面卡一处弱投影（书的实体隐喻），
不再新增 box-shadow 卡片；hover 用描边加深/轻缩放，不做 translateY 抬升。

**空状态三段式**：为什么空 → 下一步怎么办 → 直达动作按钮（书架空态、搜索无结果均已落地）。

**错误文案**：发生了什么 + 用户能做什么 + 边界值；禁 emoji/感叹号/责备语气/暴露技术码
（搜索失败与无结果文案已按此重写）。

**触达目标**：可点击元素 ≥ 44×44（翻页条、底部操作栏已对齐）。

**间距**：8px 基准梯度（4/8/12/16/24/32），卡片内 12–24、卡片间 16–24、区块间 32+。

## 9. v1.2 增补：入场动效体系（借鉴 ReactBits 范式 + UI Skills 可访问性原则）

动效基建统一在 `app/lib/ui/motion.dart`，三个原语：

- **Reveal**：透明度 + 12px 上移 + 可选模糊消散（BlurText 范式），400ms `cubic(0.16,1,0.3,1)`
- **staggerDelay(i)**：列表交错入场，步长 70ms、封顶 10 步（长列表后段不等待）
- **CountUp**：数字滚动至终值，700ms easeOutCubic（统计页数值）

落点：欢迎页品牌区（模糊渐显 120ms 步长交错）、书架卡片首载交错（重建不重播）、
统计页 tile 交错 + 数值滚动 + 热力图渐显、阅读器章节切换 260ms 淡入轻移。

硬规则：所有动效必须尊重系统「减弱动态效果」（`MediaQuery.disableAnimations`），
开启时直出终态零动画——已由单测锁定。动效仍只出现在状态变化时刻，禁循环/纯装饰。

来源说明：ReactBits / UI Skills 为 React 生态资源，本项目为 Flutter——
只移植其动效范式与设计原则，不引入任何 React 代码；图标体系维持 Material 不混装。
